# IntelliJ IDEA Keybindings

这个插件可以让你在vscode种使用 idea 的快捷键，无缝切换。

插件地址：IntelliJ Importer (opens new window)

标签高亮匹配插件，选中标签，另一半的标签将会高亮

效果展示：

插件地址：Highlight Matching Tag (opens new window)

插件地址：IntelliSense for CSS class names in HTML (opens new window)

插件地址：Auto Close Tag (opens new window)

效果展示：

插件地址：Auto Rename Tag (opens new window)

# Auto Import - ES6, TS, JSX, TSX
